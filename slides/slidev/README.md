# Slidev Deck

Presentation:

`Multimodal Visual Information Retrieval of Traditional Portuguese Instruments: A Reproducible Comparison of Dense Retrieval and Agentic Reranking`

## Data Source

The deck uses the GPU full-run snapshot in:

`../../results/esalab-big/2026-06-30_gpu_full/`

The copied chart asset `assets/recall_at_k.png` comes from:

`../../results/esalab-big/2026-06-30_gpu_full/outputs/reports/figures/recall_at_k.png`

## Commands

```bash
npm install
npm run dev
npm run build
npm run export
```

`npm run build` writes static HTML to `dist/`.
`npm run export` writes a PDF to `dist/portuguese-instruments-ir.pdf`.
